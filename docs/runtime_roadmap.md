# Runtime Roadmap

Dopo il completamento del configuration assembler il progetto proseguirà con i seguenti moduli.

---

# Entity Registry

Responsabilità:

- costruzione grafo entità
- indicizzazione entità
- gestione dipendenze

---

# State Engine

Responsabilità:

- mantenimento stato entità
- propagazione aggiornamenti
- valutazione entità derivate

---

# MQTT Bridge

Responsabilità:

- ricezione messaggi
- aggiornamento stato
- pubblicazione comandi
